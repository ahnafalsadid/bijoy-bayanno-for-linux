# 🇧🇩 Bijoy Bayanno for Ubuntu / Linux

[![Platform](https://img.shields.io/badge/platform-Ubuntu%20%7C%20Debian%20%7C%20Linux%20Mint-orange.svg)](https://ubuntu.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An automated, hassle-free installer that configures the **Bijoy Classic** and **Bijoy Unicode** keyboard layouts using IBus `m17n` on Linux distros. 

This repository also automatically bundles and installs **essential Bangla fonts** (including *SutonnyMJ* variations) required for rendering Bijoy text perfectly.

---

## ⚡ Quick Installation

You do not need to download zip files or manually move layouts around. Open your terminal (**Ctrl + Alt + T**) and run this single command:

```bash
curl -sSL [https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME/main/install.sh](https://raw.githubusercontent.com/ahnafalsadid/bijoy-bayanno-for-linux/main/install.sh) | bash
```

- Good bye
