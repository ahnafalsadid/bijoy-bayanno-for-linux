### Follow the steps to setup
- Download the Source(ZIP) file.
- Extract it.
- Open terminal inside that extracted folder.
- Run this command
- ```bash
  bash install.sh
  ```
- Install required fonts from fonts folder.
- Add keyboard layout on your linux settings.
